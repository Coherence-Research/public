# Standards

Canonical specifications published by Coherence Research.

Standards are **formal, closed, and normative**. Each standard is a precisely bounded specification — not a framework, methodology, or toolkit. Once published, a version is immutable. Updates produce new versioned releases.

All standards are released under [CC BY-ND 4.0](../LICENSE).

---

## Published Standards

### SC-AS — Structural Coherence Anchor Specification

The foundational standard. Defines the minimal structural axioms and constraints that any coherent system must satisfy.

| Document | Role | Version |
|---|---|---|
| [SC-CORE-000001](SC-CORE/v1.0/) | Core structural specification | v1.0 |
| [SC-AXIOM-000001](SC-AXIOM/v1.0/) | Axiomatic foundations | v1.0 |
| [SC-HDR-000001](SC-HDR/v1.0/) | Canonical document header schema | v1.0 |
| [SC-SCOPE-000001](SC-SCOPE/v1.0/) | Scope and admissibility boundary | v1.0 |

The SC-AS canonical set is **closed** — no external document may extend, override, or enter it.
